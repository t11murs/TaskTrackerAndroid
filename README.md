# TaskTrackerAndroid

Android-приложение для управления задачами.  
Учебный проект, демонстрирующий архитектуру Android-приложения и работу с состоянием, экранами и бизнес-логикой.

---

## Описание проекта

TaskTracker — это мобильное приложение для создания и отслеживания задач.  
Проект реализован с использованием архитектурного подхода **MVVM**, что обеспечивает разделение UI, логики и данных.

Приложение позволяет:
- создавать и просматривать задачи;
- управлять состоянием задач;
- работать с данными через отдельный слой сервисов.

## Архитектура

Проект построен по принципам **MVVM**:

- **Activity** — хост экранов и навигации  
- **Screen** — UI-компоненты  
- **ViewModel** — бизнес-логика и состояние  
- **Model** — модели данных  
- **Service** — работа с источниками данных  
- **Theme** — стили и оформление приложения
